# Business-Analytics-Projects
Business Analytics Projects

This repository contains two business analytics projects.

Project 1: Customer Churn Prediction
This project predicts customer churn using machine learning and provides business insights through an Excel dashboard.

Project 2: Market Basket Analysis
This project analyzes customer purchase behavior using association rule mining to identify product combinations.

Tools Used:
- Python
- Excel
- Power BI
